# Hoang Nhan Nano Biotech

Welcome to the official repository for the **Hoang Nhan Nano Biotech** website. This platform showcases our sustainable and advanced nanobiotechnology solutions for agriculture, crop protection, and plant health.

[https://www.nanobiotech.com.au](https://www.nanobiotech.com.au)

## Overview

Hoang Nhan Nano Biotech focuses on sustainable solutions for crop protection and enhancement of crop yields nationwide and globally. We repurpose agricultural waste and natural resources into cutting-edge nano biomaterials to deliver targeted plant nutrition and immunity-boosting solutions.

## Tech Stack

This website is built with modern web technologies:

- **[Astro](https://astro.build/)** - A fast, content-focused web framework.
- **[Tailwind CSS](https://tailwindcss.com/)** - A utility-first CSS framework for rapid UI development.
- **TypeScript** - For scalable and robust code.

## Getting Started

### Prerequisites

- Node.js (version `^18.17.1 || ^20.3.0 || >= 21.0.0`)
- npm

### Installation

1. Clone the repository:

   ```shell
   git clone https://github.com/lenhanpham/nanobioweb.git
   cd nanobioweb
   ```

2. Install dependencies:
   ```shell
   npm install
   ```

### Development

Start the local development server:

```shell
npm run dev
```

The site will be available at `http://localhost:4321`.

### Build & Deploy

Build the production-ready site:

```shell
npm run build
```

This project is configured to use the Cloudflare adapter (`@astrojs/cloudflare`) for deployment.

## Code Quality

Run tests, linting, and formatting checks to ensure code quality:

```shell
npm run check
```

Automatically fix linting and formatting issues:

```shell
npm run fix
```
